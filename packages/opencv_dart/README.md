# opencv_dart

opencv for Flutter, included modules:

- calib3d
- core
- contrib
  - aruco
  - img_hash
  - wechat_qrcode
  - ximgproc
  - xobjdetect
  - quality
- dnn
- features2d
- highgui
- imgcodecs
- imgproc
- objdetect
- photo
- stitching
- video
- videoio

If `highgui` or `videoio` is NOT required, use [opencv_core](https://pub.dev/packages/opencv_core)
